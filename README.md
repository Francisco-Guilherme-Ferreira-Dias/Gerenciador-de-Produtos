# 🛒 Gerenciador de Produtos

Sistema de gerenciamento de produtos desenvolvido com **Node.js**, **Express** e **SQLite**, criado para praticar habilidades em desenvolvimento web fullstack com API REST e banco de dados.

---

## 🚀 Funcionalidades

- **Cadastrar** novos produtos
- **Listar** todos os produtos
- **Atualizar** dados de um produto
- **Remover** produtos do sistema
- Interface web simples com **HTML, CSS e JavaScript**

---

## 🛠️ Tecnologias utilizadas

**Front-end**
- HTML
- CSS
- JavaScript

**Back-end**
- Node.js
- Express

**Banco de dados**
- SQLite

---

## ▶️ Como rodar

### Pré-requisitos
- Node.js instalado

### Passos

```bash
# Clone o repositório
git clone https://github.com/Francisco-Guilherme-Ferreira-Dias/Gerenciador-de-Produtos.git

# Entre na pasta do back-end
cd Back

# Instale as dependências
npm install

# Inicie o servidor
node server.js
```

Acesse no navegador: `http://localhost:3000`

---

## 📚 O que aprendi

- Criação de uma **API REST** com Node.js e Express
- Integração com banco de dados **SQLite**
- Consumo de API com **fetch** no front-end
- Comunicação entre **front-end e back-end**
- Operações **CRUD** completas com banco de dados relacional
